# Gems Auto-Saver 使い方ガイド

このツールは、Gemini Gems の回答から「スライド（HTML/CSS）」や「画像」を自動的に抽出し、ローカルに保存するためのブラウザ用スクリプトです。

## 使用手順

### 1. 準備
1. Gemini (Gems) のページを開きます。
2. 保存したい回答が生成される状態、またはこれからプロンプトを投げる状態にします。

### 2. ツールの起動
1. ブラウザで **F12** キーを押して「デベロッパー ツール」を開きます。
2. 「**コンソール (Console)**」タブを選択します。
3. 私がこれから提供する JavaScript コードをすべてコピーし、コンソールに貼り付けて **Enter** キーを押します。

### 3. 操作方法
画面右上に **[Gems Auto-Saver]** というネオンパネルが表示されます。
- **Prompt:** Gems に投げたい指示を入力します。
- **Execute & Save:** ボタンを押すと自動で送信され、回答が終わるのを待ってから保存ダイアログが出ます。
- **Manual Save:** すでに生成済みの回答を保存したい場合は、このボタンを押すと最新の成果物を抽出します。

## 保存されるもの
- **スライド:** 回答内の HTML/CSS コードブロックを 1 つの `.html` ファイルにまとめます。
- **画像:** 生成された画像がある場合、個別にダウンロードされます。
- **テキスト:** 回答の全文を `.txt` として保存します。
