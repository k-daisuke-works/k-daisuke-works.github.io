# 実装計画書：ツール統合とUI統一

## 1. 目的
バラバラに存在していた4つのWEBツールを1つの「統合ダッシュボード」にまとめ、操作性を向上させるとともに、最新のプレミアムデザイン（ダークモード、ガラスモーフィズム）に統一します。

## 2. 統合対象ツール
1. **シミュレーション結果生成**: テキストベースの進捗記録からスライドを生成。
2. **評価点グラフ分析**: CSVから社会福祉士とAIの比較チャートを生成。
3. **事例スライド(プレゼン)**: 画像ベースのスライドショー表示。
4. **事例スライド(配布用)**: テキスト事例からA4印刷・スライド用レイアウトを生成。

## 3. デザイン方針
- **カラーパレット**: 
  - 背景: 深い紺色 (#0f172a / #020617)
  - アクセント: 信頼感のあるブルー (#3b82f6) と ゴールド (#fbbf24)
- **タイポグラフィ**: 
  - 英数字: Outfit / Inter
  - 日本語: Noto Sans JP
- **視覚効果**: ガラスモーフィズム（透過・ぼかし背景）、スムーズなフェード遷移。
- **統一レイアウト**: 左サイドバー（ナビゲーション）＋右メインコンテンツエリア。

## 4. 技術的アプローチ
- **SPA構造**: JSによる表示切り替えを採用し、1つのHTML内で完結させます。これにより、ツール間の移動を高速化します。
- **モジュール化**: 各ツールのロジックは独立した関数・オブジェクトとして定義し、名前の競合を防ぎます。
- **外部ライブラリ**: `Chart.js` をCDNから一括で読み込みます。

## 5. 実装ステップ
1. **ディレクトリ整理**: 共通ファイルを `まとめ/` 直下に配置。
2. **スタイル実装**: 高度なCSS変数を用いた `master.css` の作成。
3. **骨格作成**: `index.html` にナビゲーションとコンテンツコンテナを実装。
4. **機能移植**: 各ツールのロジック（パーサー、チャート描画、スライド制御）を1つずつ `app.js` に統合。
5. **検証**: 各ツールの機能が損なわれていないか個別チェック。
