# 実装計画 - 評価点比較レーダーチャート

## 1. デザインコンセプト
- **プレミアム感**: ダークテーマを基調とし、ネオン調のアクセントカラー（社会福祉士は青、生成AIはピンク/紫）を使用する。
- **視覚的効果**: グラフの背景にグラデーションを使い、カードスタイルで情報を整理する。
- **フォント**: Google Fontsから "Inter" を採用し、クリーンな印象を与える。

## 2. 実装ステップ

### ステップ 1: 基盤作成
- `index.html` の作成
- `style.css` で基本スタイルの定義
- Chart.js の CDN リンクを追加

### ステップ 2: CSVパース処理
- ファイル選択 (`<input type="file">`) 機能を実装。
- `FileReader` を使用してCSVデータを取得。
- 文字列を分割して、5つの評価項目ごとの平均値（または代表値）を計算するロジックを実装。
- CSVのカラム名に含まれる `[社会福祉士]` と `[生成AI]` を判別してデータを分類。

### ステップ 3: グラフ描画
- Chart.js を使用してレーダーチャートを描画。
- 5角形（論理的根拠、実現可能性、倫理基準、寄り添い、好み）の軸を設定。
- データの重ね合わせを調整（透明度など）。

### ステップ 4: ブラッシュアップ
- アニメーションの追加。
- データの要約表示（数値一覧）。
- エラーハンドリング（不正なCSV形式など）。

## 3. データの紐付け
CSVのヘッダーから以下の項目を抽出：
1. 論理的な根拠
2. 現場での実現可能性
3. 倫理基準
4. 要支援者への寄り添い
5. 好み
