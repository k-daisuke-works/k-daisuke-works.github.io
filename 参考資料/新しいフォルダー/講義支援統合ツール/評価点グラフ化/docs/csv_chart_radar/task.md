# タスク: CSVデータを基にした社会福祉士と生成AIの評価点比較レーダーチャートの作成

## 概要
CSVファイルを読み込み、「社会福祉士」と「生成AI」の5つの評価項目を比較する五角形のレーダーチャートを表示するHTMLアプリケーションを作成する。

## 完了条件
- [ ] CSVファイルを読み込めること
- [ ] 「社会福祉士」と「生成AI」のデータを抽出し、比較できるようにすること
- [ ] Chart.jsを使用して、五角形のレーダーチャートを描画すること
- [ ] モダンでプレミアムなデザイン（ダークモード、グラデーションなど）を適用すること
- [ ] レスポンシブ対応であること

## 使用技術
- HTML5 / CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
- Chart.js (グラフ描画ライブラリ)
- PapaParse (CSVパース用ライブラリ、または標準JSでのパース)
