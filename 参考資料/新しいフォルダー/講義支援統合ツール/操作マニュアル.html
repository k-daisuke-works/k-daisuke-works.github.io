事例検討 AIシミュレーション
一連の操作マニュアル

このマニュアルは、Social Work AI Portal内の各ツールや生成AIを活用し、事例の生成から講義・シミュレート・最終評価（グラフ化）までをスムーズに実施するための、全体の操作フローをまとめたものです。

🕒 フェーズ1：事前準備と教材の作成（講義開始前）

1. ツールの起動と事例の準備

  1. フォルダ内にある バッチファイル を起動し、ポータル画面を開きます。
  2. AI等を用いて事例のテキストデータを生成し、全体をコピーします。
  3. コピーしたテキストを Gems連携ツール に貼り付けて実行します。
    （※ここから得られた情報を元に、後続の資料を作成していきます）

2. 受講者向け資料の出力

  1. ポータルの 配布・印刷資料生成 ツールを開きます。
  2. 情報を入力してPDFを生成します。
  3. 「A4印刷版」をクリックし、受講者への配布用に印刷を行います。

3. プロジェクター用スライドの準備

  1. AIから生成されたジェノグラム（家系図）やスライド用のHTMLコードをコピーします。
  2. ポータルの HTML反映ツール にコードを貼り付けてプレビューを実行します。
  3. 表示された画面の スクリーンショット（SS） を撮影して保存します。
    （※合わせて、Gemsが生成した画像素材なども保存しておきます）
  4. ポータルの プレゼンスライド ツールを起動します。
  5. 用意したスクリーンショット画像やその他の画像を、画面にドラッグ＆ドロップで追加します。
  6. 「スライドショーを開始」ボタンを押し、プロジェクターに投影します。
🗣️ 〜 しばらく講義を実施（この講義時間中に、裏で生成AIの回答を前もって生成しておきます）〜

🔄 フェーズ2：講義とシミュレーションの実施

4. グループワーク・議論の集約（各対応フェーズ）

  1. グループワーク等の音声を議事録（テキスト）に変換します。
  2. テキストをAIに読み込ませ、「長期目標」「具体的な行動」 の2点に整理させます。
  3. 整理されたテキストデータを「事例生成AI」に読み込ませ、シミュレーション結果（テキスト）を出力させます。
  4. そのテキストを、ポータルの シミュレート記録 ツールの「初期対応（または中間・最終対応）」に入力してスライドを生成します。
  5. 生成されたスライドを印刷（PDF化）し、オープンチャット等へ貼り付けて受講者に共有します。
⚠️ 〜 受講者の待機時間中に裏で生成AIの回答生成も忘れずに行い、「初期対応 → 中間対応 → 最終対応」と最後までリピートします 〜

🏁 フェーズ3：総括と評価・振り返り（講義終盤）

5. 全体のまとめと振り返り

  1. 最終対応が終わった後、すべてのシミュレート結果テキストをまとめます。
  2. ポータルの 事例総括生成 ツールにまとめて読み込ませます。
  3. 生成された全体像の総括スライドをPDFで配布、およびスライドショーで確認します。
  4. 「生成AIが考えたバージョンの対応経路」も同様に作成し、スライドショーとして投影・比較して見せます。

6. 評価・フィードバックの回収と分析

  1. Googleフォーム等のアンケートを用いて、受講者からの投票や評価を実施します。
  2. フォームの結果データを CSV形式 でダウンロード（出力）します。
  3. 出力したCSVファイルを、ポータルの 評価点比較分析 ツールに直接読み込ませます。
  4. 画面に表示されたレーダーチャート等のグラフや結果データをプロジェクターに映します。
  5. 受講者からのコメントなどを読み上げ、全体の講評・まとめを行って講義を終了します。
