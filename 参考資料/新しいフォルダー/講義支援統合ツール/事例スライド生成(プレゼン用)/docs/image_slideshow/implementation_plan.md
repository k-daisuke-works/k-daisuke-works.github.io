# 実装計画 - 閉じるボタンの追加

UIに「閉じるボタン」を追加し、クリック時にブラウザウィンドウを閉じる機能を実装します。

## 変更内容

### 1. HTML (`index.html`)
- `action-bar` 内に「閉じる」用のボタンを追加します。
- 分かりやすいように「×」のSVGアイコンを使用します。

### 2. CSS (`index.css`)
- 閉じるボタンのスタイルを調整します。
- 他のアクションボタンと統一感を持たせつつ、ホバー時に削除や終了を想起させる色味（赤系）の微かなフィードバックを追加します。

### 3. JavaScript (`script.js`)
- 閉じるボタンのクリックイベントを実装します。
- `window.close()` を呼び出して、ブラウザを閉じる処理を行います。

## 留意点
- ブラウザのセキュリティ設定により、スクリプトで開かれたウィンドウ以外は `window.close()` が動作しない場合があります。
- プレミアムなUIを維持するため、アイコンのデザインやホバーアニメーションを丁寧に調整します。
