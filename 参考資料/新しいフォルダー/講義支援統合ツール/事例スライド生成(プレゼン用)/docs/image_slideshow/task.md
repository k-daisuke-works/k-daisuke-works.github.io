# タスク: 画像スライドショーWebアプリの作成

## 概要
ユーザーが選択した画像（特にファイル名に「①」などが含まれるもの）を、正しい順序でスライドショー形式で表示するWebアプリケーションを作成する。

## 要件
- [ ] HTML/CSS/JSによる実装
- [ ] プレミアムでモダンなデザイン
- [ ] ファイル名の数字（①、②...や1, 2...）を認識したソート機能
- [ ] スライド切り替え機能（前へ/次へ/オートプレイ）
- [ ] フォルダまたは複数ファイルの一括読み込み機能
- [ ] レスポンシブデザイン
