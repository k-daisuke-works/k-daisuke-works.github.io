# タスク：テキストファイル読み込み・自動スライド構築機能の実装

## 概要
ユーザーから提供された固定構造のテキストファイル（事例.txt等）をブラウザ上で読み込ませ、JavaScriptで自動構造化してスライドUIに流し込むツールを作成する。

## 目標
1. **ファイル読み込みUI**: HTML上にドラッグ＆ドロップまたはファイル選択ダイアログでテキストをブラウザにロードできるインターフェースを設けること。
2. **自動パース（解析）処理**: 読み込んだテキストから、「【事例：xxx】」「・ケース概要」等の固定見出しをフックとして各セクションに文章を分類する機能をJavaScriptで実装する。
3. **動的スライド生成**: 抽出したテキストを、作成済みのグラスモーフィズムデザインのカードUIに自動展開し、2部構成のスライドとして表示させる。
